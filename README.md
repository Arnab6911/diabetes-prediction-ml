# Diabetes Prediction using Machine Learning

## Project Overview
This project predicts whether a person has diabetes based on medical attributes using machine learning algorithms.

## Dataset
- Pima Indians Diabetes Dataset

## Algorithms Used
- K-Nearest Neighbors (KNN)
- Decision Tree

## Steps Performed
- Exploratory Data Analysis (EDA)
- Data preprocessing and feature scaling
- Model training and evaluation

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Result
Both models were evaluated using accuracy, confusion matrix, and ROC curve.
